# Survey Insight

将原始 CSV 数据转化为精美、交互式的仪表盘，并提供 AI 驱动的定性分析。

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `MINIMAX_API_KEY` in [.env](.env) to your MiniMax API key
3. Run the app:
   `npm run dev`
